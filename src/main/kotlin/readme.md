# WiamVideoMaker - 自动字幕生成器

## 简介

WiamVideoMaker 是一个自动字幕生成工具，旨在帮助我自己快速为视频生成字幕。

## 功能特性

- 使用 AssemblyAI API 识别文字
- 没有图形界面
- 按句子划分字幕
- 准确率一言难尽
- 注意一下 API 密钥，**请一定要**截图的时候泄露出去
- 字幕格式是 .ass，复制粘贴到 Aegisub 就可以了
- 生成的字幕样式是 ENSub1 ，不想要的话生成出来后自己改就行
- 实测最好模型跟通用模型效果一样，如果你钱多得没地烧就用最好模型

## 安装与使用

1. 确保已安装 Java
2. 用这玩意运行 ```java -jar WiamVideoMaker-1.0-SNAPSHOT.jar```
3. 按提示操作就行
4. 先上传，再识别，最后转换
